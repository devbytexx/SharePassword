# Screenshots

Diese Bilder werden im Haupt-[README](../README.md) eingebunden. Lege hier PNGs
mit genau diesen Namen ab, dann erscheinen sie automatisch:

| Dateiname      | Inhalt                                                     |
| -------------- | ---------------------------------------------------------- |
| `create.png`   | Startseite / Geheimnis-Erstellen-Formular (mit Datei-Upload) |
| `preview.png`  | Geöffnetes Vorschau-Modal (z. B. Bild oder PDF)            |
| `view.png`     | Empfänger-Ansicht mit entschlüsseltem Inhalt + Dateiliste   |

**Empfehlung:** Breite ca. 1200–1600 px, als PNG speichern, auf < 500 KB pro
Bild halten (z. B. mit [squoosh.app](https://squoosh.app)). Zum Aktualisieren
einfach unter gleichem Namen überschreiben.
